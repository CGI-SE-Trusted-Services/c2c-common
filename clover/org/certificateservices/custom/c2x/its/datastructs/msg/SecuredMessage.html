33    * of the variable header, payload and trailer fields. A message that does not conform to the profile is invalid. The
34    * default value shall be set to 0, if no specific profile is used. Not used for version 2 messages.
35    *
36    * <li> header_fields is a variable-length vector that contains multiple information fields of interest to the
